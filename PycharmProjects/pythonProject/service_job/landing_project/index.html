Проблема?

Сейчас вся информация, представленная на сайте, выводится из базы данных. И для управления базой данных мы обращаемся к административной части сайта. Соотвественно необходимо, чтобы интерфейс административной части чайта был прост и удобен в использовании, чтобы можно было с легкостью структуризировать информацию, добавлять ее, редактировать, удалять и т.д. С помощью Django мы можем реализовать все эти функции, для удобства также можно добавить фильтры, поиск, что я и сделала.

Используемые технологии

Django

Фреймворк, подходящий для разработки сложных сайтов и веб-приложений, написанный на языке программирования Python. С помощью этого фреймворка осуществлялась реализация и настройка административной части, управление базой данных.

HTML

Написание самого лендинга проекта, написание статичсекого кода для главной страницы. Разметка страницы и расположение блоков.

CSS5

Написание стилей для страницы. Разработка внешнего оформления страницы и отображения элементов.

Фреймворк Materialized

Использование такого фреймворка гораздо упрощает верстку и создание адаптивного интерфейса.

Postman

Удобный HTTP-клиент для тестирования веб-сайтов. С помощью этого инструмента осуществлялось тестирование RESTfull API.

Django REST framework

Реализация RESTfull API. Работа с форматом json.
